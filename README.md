# Ko-RSS-Scraper

각종 한국 신문사의 RSS를 기반으로 원하는 카테고리의 뉴스 구독 서비스를 제공하는 Nest.js 기반 API

## 개발환경

### 프레임워크

Nest.js

### 데이터베이스

SQLite

### ORM

Prisma
